void RCReceiver::trigger_signal ( )

Triggers the signal capture for the RC receiver.

This function is called to handle a direct input capture. It captures the RC signal pulse width and updates the signal value. It also checks to make sure the signal doesn't go out of sync, or to not use a bad signal.
